# MySQL Demo

A basic MySQL program demo written in Visual Basic that prints out records from the sakila database that is included in MySQL Workbench.

## Usage

To run this, first execute the SQL script stored at `SQL Scripts/sakila_actor.sql` in the solution against your MySQL instance to create and populate the database objects required for this demo, if necessary. After the script has completed successfully, the project will be ready to use.

When running the program, provide the login credentials for your MySQL instance, and the console will output the first 10 records from the table generated by the script.
